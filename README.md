# Pixel Nav

**Pixel Nav** es un videojuego desarrollado en Python utilizando Pygame, inspirado en los clásicos juegos de naves espaciales de los años 80.

El jugador controla una nave que se mueve horizontalmente y debe destruir enemigos, evitar obstáculos y recolectar potenciadores para sobrevivir el mayor tiempo posible.

---

## Controles

* ⬅️ / ➡️ → Mover nave
* ESPACIO → Disparar
* ESC → Salir / Pausar

---

##  Cómo ejecutar el proyecto

1. Tener Python instalado
2. Instalar Pygame:

```bash
pip install pygame
```

3. Ejecutar:

```bash
python main.py
```
